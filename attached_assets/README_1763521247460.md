
# YoRight 🌙✈️

**YoRight** is a luxury OTA (Online Travel Agency) built for the Saudi market, featuring a world-class native mobile experience (iOS & Android) and a robust Node.js gateway.

## 🏗 Architecture

*   **Mobile:** Native iOS (SwiftUI) & Native Android (Compose). Offline-first, Google Maps, RTL-first.
*   **Gateway:** NestJS (Fastify) acting as a Backend-for-Frontend (BFF) and Aggregator.
*   **Data:** PostgreSQL (Prisma), Redis (Caching).
*   **Suppliers:** RateHawk (Hotels), Tap (Payments).

## 🚀 Getting Started

### Prerequisites
*   Node.js 20+
*   Docker & Docker Compose
*   Xcode 15+ (for iOS)
*   Android Studio Koala+ (for Android)
*   Ruby (for Fastlane)

### Quick Start (Monorepo)

```bash
# 1. Bootstrap dependencies and env vars
./tools/scripts/bootstrap.sh

# 2. Start Local Stack (DB, Redis)
docker-compose up -d

# 3. Seed Database (Cities, Hotels, Admin)
cd apps/gateway && npm run seed

# 4. Start Gateway
cd apps/gateway && npm run start:dev
```

## 🌍 Localization
*   **Arabic (ar-SA):** Default. Uses Arabic-Indic numerals.
*   **English (en-US):** Optional.

## 📱 Native Features
*   **Deep Links:** `yrt://hotel/:id`, `yrt://search?city=...`
*   **Maps:** Google Maps SDK with clustering.
*   **Payments:** Apple Pay, mada, Credit Cards (via Tap).

## 🧪 Testing
*   **Gateway:** `npm test` (Unit), `npm run test:e2e` (Integration).
*   **iOS:** `Cmd+U` in Xcode.
*   **Android:** `./gradlew test connectedAndroidTest`.

## 📦 CI/CD
GitHub Actions pipeline builds, tests, and uploads artifacts. Fastlane handles store submission.
